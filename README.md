# 鉄板ダイニングおっとり新町店 - ビールサブスク自動返信システム

## 📌 概要
本システムは、LINE公式アカウントのMessaging APIと連携し、お客様から特定のキーワード（「テスト」等）を受信した際、ビールサブスクの電子会員証を自動応答で即座に補給・発行するプロ仕様のバックエンドシステムです。

## 🛠️ 技術スタック
- **実行環境:** Node.js
- **フレームワーク:** Express
- **SDK:** @line/bot-sdk
- **インフラ:** Render (Cloud Hosting)

## 🔒 セキュリティおよび稼働ロジック
- Googleのマルチログインバグや、システムによる「302 Found」のリリダイレクト制限を完全に回避する独立サーバー構成。
- LINE Developersからの検証用GETリクエストに対し、常時ステータス「200 OK」を返却する防壁ルートを確立。
- 5月の本番リリースに向けた、ホットペッパー販促連動型の高可用性サブスクリプションモデルに対応。
